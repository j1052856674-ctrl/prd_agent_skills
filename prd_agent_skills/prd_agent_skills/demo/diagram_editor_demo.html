功能介绍

这是一个强大的Mermaid图表编辑器，支持多种图表类型的实时编辑和预览。

  • 支持流程图、序列图、类图、状态图、ER图、甘特图、饼图
  • 实时预览，语法错误提示
  • 撤销/重做功能
  • 支持导出为图片和文本
  • 提供多种模板快速开始

💡 使用提示

  • 快捷键： Ctrl/Cmd + S 保存图表，Ctrl/Cmd + Z 撤销
  • 编辑： 点击编辑器开始编辑代码，图表会实时更新
  • 验证： 点击"验证"按钮检查语法是否正确
  • 模板： 使用"插入模板"快速生成图表结构
  • 导出： 可以导出为PNG图片或文本文件
