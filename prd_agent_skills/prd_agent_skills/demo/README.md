# PRD Generator 演示页面

本目录包含PRD Generator的各种演示页面和工具展示。

## 演示页面

1. **main_demo.html** - 主演示页面
   - 展示PRD Generator的核心功能和特性
   - 包含功能介绍和技术栈说明

2. **simple_demo.html** - 简化版演示
   - 精简的功能展示
   - 快速了解基本功能

3. **diagram_editor_demo.html** - 架构图编辑器演示
   - 展示Mermaid图表编辑功能
   - 支持多种图表类型

4. **text_editor_demo.html** - 文本编辑器演示
   - 展示实时编辑功能
   - 支持点击编辑、自动保存

5. **import_demo.html** - HTML导入演示
   - 展示从设计稿生成原型
   - 支持图片和HTML文件导入

6. **test_page.html** - 测试页面
   - 功能测试和验证

## 示例文件

1. **data_governance_prd.md** - 数据治理系统PRD示例
2. **test_prd_generation.md** - 测试生成的PRD示例

## 旧版本工具

1. **html_importer.js** - 旧版HTML导入器
2. **text_editor.js** - 旧版文本编辑器
3. **diagram_editor.js** - 旧版架构图编辑器
4. **simple_text_editor.js** - 简化文本编辑器
5. **simple_html_importer.js** - 简化HTML导入器

## 使用说明

直接在浏览器中打开任意HTML文件即可查看演示效果。
