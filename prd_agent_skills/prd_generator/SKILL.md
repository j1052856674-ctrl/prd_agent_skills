## 🎯 双轨制市场调研模式详解

### 数据源配置
- **市场分析轨道**：
  - A级：国家统计局、教育部、工信部（.gov.cn）
  - B级：艾瑞咨询、QuestMobile、头豹研究院（.org）
  
- **功能调研轨道**：
  - A级：知乎专业回答、小红书用户体验
  - B级：GitHub开源项目、产品官网

### 搜索策略
- **市场分析**：WebSearch + 精确匹配官方报告
- **功能调研**：exa_search + 固定站点搜索

### 报告结构
- **市场分析报告**：规模数据、用户特征、政策环境
- **功能调研报告**：竞品对比、技术实现、用户体验

## 📝 核心原则

1. **进度可视化**：每个步骤都要显示进度条
2. **强制用户确认**：每个步骤都必须等待用户明确确认后才能进入下一步
3. **具体化执行**：说明具体在做什么，使用什么数据
4. **结构化输出**：使用标准PRD格式
5. **双轨制调研**：明确区分市场分析和功能调研
6. **本地优先**：所有功能基于本地能力实现
7. **步骤间严格分离**：避免自动跳过任何步骤
