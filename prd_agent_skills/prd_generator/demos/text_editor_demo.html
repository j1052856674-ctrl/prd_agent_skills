📝 简单文本编辑

点击下面的文本开始编辑。支持快捷键：

  • Enter - 保存编辑
  • Esc - 取消编辑
  • Ctrl/Cmd + Z - 撤销
  • Ctrl/Cmd + Y - 重做
  • Ctrl/Cmd + S - 保存所有
在这里点击开始编辑这段文字。编辑时会自动显示格式工具栏，支持基本的富文本编辑功能。

这是一个富文本编辑示例，你可以：

📝 代码和特殊内容

编辑器会保留代码块的格式：

// 这是一个代码示例
function greet(name) {
    return "Hello, " + name + "!";
}

// 调用函数
console.log(greet("World"));
你可以在上面编辑代码，编辑器会尝试保留代码的格式。但是请注意，富文本编辑器可能会改变代码的缩进。

💾 导出和导入

你可以导出所有编辑的内容，之后可以重新导入恢复。
