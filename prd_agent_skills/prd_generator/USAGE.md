# 使用指南

这个Skill可以直接在Claude Code中使用。

## 触发方式

直接告诉它你想做什么，例如：
- "帮我做一个电商小程序的PRD"
- "生成一个PRD"
- "写产品需求文档"

## 注意事项

1. **无需配置**: 不需要额外配置API Key，直接使用当前Claude Code模型
2. **在对话中生成**: 所有PRD内容都在对话中直接生成，不需要外部调用
3. **结构化输出**: 生成标准化的PRD文档，包含完整的章节

## PRD包含内容

- 项目信息（名称、类型、目标用户、优先级）
- 需求背景（现状痛点、业务目标、用户价值）
- 需求目标（核心目标、次要目标、衡量指标）
- 需求概述（产品定位、核心功能、用户场景）
- 详细方案（按模块详细描述）
- 异常处理（权限、超时、敏感数据等）
- 上线计划（分阶段上线、回滚策略）

## 可选功能

生成PRD后可以：
- 导出为Markdown格式
- 导出为HTML格式
- 查看交互原型（文字描述）
- 查看版本历史
- 对比版本差异
- 修改特定章节
